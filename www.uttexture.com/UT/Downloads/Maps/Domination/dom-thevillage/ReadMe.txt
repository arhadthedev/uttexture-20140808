Map Overview:

This is a small Domination map with 5 control points. Due to the
size og the map it will bring instant action even with little
player load. The map supports all weapons. The map is fully bot
supported.


Installation:

Copy the DOM-(AMLP)TheVillage.unr into your UT Map folder
Copy the TobioTrees.u file into your UT system folder

then launch UT and select the map, enjoy!
