Installation


Unzip the DOM-Arena16LE's *.unr file in your Unreal Tournament/Maps directory, then either doubleclick on it, or type 'open DOM-Arena16LE ' in the console or select the Map in a Botmatch or an Internet game.

*.unr files go in Maps/
*.utx files go in Textures/
*.umx files go in Music/
*.uax files go in Sounds/
*.u and *.int files go in System/

Author's Notes


I customized DM-Arena16 for online tournament play, specifically for Proving Grounds, although others can use this in their DOM servers for a futuristic alternative to all the castle DOMs.

Also, I made this architecture simple enough that ten seconds after you enter the map, you pretty much know where everything is!

Enjoy!
