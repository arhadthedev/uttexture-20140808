Installation


Unzip the DOM-ZHaste's *.unr file in your Unreal Tournament/Maps directory, then either doubleclick on it, or type 'open DOM-ZHaste ' in the console or select the Map in a Botmatch or an Internet game.

*.unr files go in Maps/
*.utx files go in Textures/
*.umx files go in Music/
*.uax files go in Sounds/
*.u and *.int files go in System/
