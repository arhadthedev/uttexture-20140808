Titre                   : La Kasbah du sud
Nom de fichier          : DOM-BTKasbah.unr
Version                 : 6 janvier 2000

Auteur                  : Bernard TEXIER
Addresse e-mail         : btexier@club-internet.fr

Description             : Map construite avec l'éditeur d'Unreal. 
                          A partir d'une vue d'une Kasbah du sud marocain
                          j'ai eu l'idée de construire cette "map" pour
                          UT. En plaçant 3 points de controle : un au dernier
                          étage (TOP), un au premier étage (MIDDLE
                          et le dernier au rez de chaussée (LOWER),
                          on dispose d'une arène pour le mode DOMINATION.


Autres niveaux UT publiés:

* Construction *

Base                    : aucune base précédante.
Temps de réalisation    : Plusieurs jours 
Editeur utilisé         : UnrealED (UT)
Bugs connus             : Aucun (je l'espère). Testée sur ma machine avec
                          UT + patch 402.
                          
Micro de realisation    : Realisé et testé avec un PII 350, SB64, 64MB
                          Carte 3DFx "TechWorks board" Voodoo 1
                          testée (efficacement) jusqu'a 16 bots avec
                          la version UT Pachs 402 (française).

* Comment utiliser ce niveau *
                        : Extraire le fichier DOM-BTKasbah.unr et le  
                          recopierdans la "directory"  ..\Maps de Tournament
                          Le fichier est initialise en Domination par défaut.
                          
                          Vous pouvez également le lancer de la manière
                          habituelle à partir du menu déroulant en mode 
                          "entrainement" (mode DOM)
                          
                          Utiliser 5 ou 7 bots. Nombre de points = 150
