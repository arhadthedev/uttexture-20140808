Release notes

Remember the amazing acrobatics in the movie Matrix ? Now you can do the same moves in Unreal Tournament with the MatrixMoves mutator. This first beta version allows you to run up the walls and do cool backflips to confuse your opponents. The effect looks really nice in both 3rd and 1st person views. But of course you have to see it for yourself !

Installation

UMOD: Uncompress the zip file and double-click the file MatrixMoves.umod
Non-UMOD: Uncompress the zip file to your UnrealTournament\System directory

How to do the moves

1. Run towards a wall.
2. When near the wall, jump towards it while looking upwards.
3. Keep on running after hitting the wall. The wallrun begins automatically.
4. During the wallrun, press the jump button to do a backflip.

How to use MatrixMoves in a network game

To use MatrixMoves mutator in a network game, server machine must have the following line in the UnrealTournament.ini file's [Engine.GameEngine] section: "ServerPackages=MatrixMoves". Note that the UMOD version of the mutator automatically adds the line to the ini file.

Known issues

- The wallrun feature works only if you play with one of the default UnrealTournament player classes.
- Bots don't use the wallrun or backflip features.
- The demo playback does not show the wallrun or backflip moves right.
