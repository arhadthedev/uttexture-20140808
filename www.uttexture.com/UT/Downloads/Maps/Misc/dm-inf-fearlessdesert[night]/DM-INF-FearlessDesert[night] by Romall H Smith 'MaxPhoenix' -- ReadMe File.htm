Installation


Unzip the DM-INF-FearlessDesert[night]'s *.unr file in your Unreal Tournament/Maps directory, then either doubleclick on it, or type 'open DM-INF-FearlessDesert[night] ' in the console or select the Map in a Botmatch or an Internet game.

*.unr files go in Maps/
*.utx files go in Textures/
*.umx files go in Music/
*.uax files go in Sounds/
*.u and *.int files go in System/


Extended Description


This map is an open battle field. Many of the items used as cover are on random spawn so you never know if you have that same hiding spot
