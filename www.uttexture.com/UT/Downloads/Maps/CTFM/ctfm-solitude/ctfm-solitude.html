Installation


Unzip the CTFM-Solitude's *.unr file in your Unreal Tournament/Maps directory, then either doubleclick on it, or type 'open CTFM-Solitude ' in the console or select the Map in a Botmatch or an Internet game.

*.unr files go in Maps/
*.utx files go in Textures/
*.umx files go in Music/
*.uax files go in Sounds/
*.u and *.int files go in System/

Author's Notes


The bots tend to use the lower level more often than the top one. However, I left this as is as the upper level dumps the attacker out right on top of the flag, and this could get really annoying if the bots used it a lot.
