Installation


Unzip the CTFM-AncientTitanV110's *.unr file in your Unreal Tournament/Maps directory, then either doubleclick on it, or type 'open CTFM-AncientTitanV110 ' in the console or select the Map in a Botmatch or an Internet game.

*.unr files go in Maps/
*.utx files go in Textures/
*.umx files go in Music/
*.uax files go in Sounds/
*.u and *.int files go in System/


Extended Description


This is an old tomb, consisting of two crypts (the bases) and a central area where a UTDMTitan.


Author's Notes


What's new in this version : I added some player starts to avoid spawndeaths ; I added water in the central area ; I built an entrance and a secret area, both without pathnodes so that bots don't go too far away from the mess ; there's a deadly trap somewhere ; a few message triggers.
