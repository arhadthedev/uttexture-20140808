UnrealEd Tutorial 5 - Adding Sound and Music by Cableless
 

This tutorial is a simple one. Adding sound and music to an Unreal level is easy using the Browser.

Download the map for this Tutorial and install it in your C:\UNREAL\MAPS directory.

 

  
 

 

Adding an Ambient Sound
 

  1. In the Browse window select Classes.
  2. In the Browse window, click on the "-" next to Keypoint to expand the list.
  3. In the Browse window select Ambient Sound.
  4. Right Click where you want to place the Ambient Sound and select Add Ambient Sound Here. You should see a key with a speaker icon attached appear in the Perspective window.
  5. Right Click on the Ambient Sound and select Ambient Sound Properties.
  6. In the Ambient Sound Properties window select Sound.
  7. In the Ambient Sound Properties window select Ambient Sound.
  8. In the Browse window select SoundFX.
  9. In the Browse window under SoundFX select General.
  10. Select Load at the bottom of the Browse window.
  11. Select AmbAncient.uax and click Open.
  12. In the Browse window select adrips4.
  13. In the Ambient Sound Properties window select Ambient Sound.
  14. In the Ambient Sound Properties window next to Ambient Sound select Use.
  15. Close the Ambient Sound Properties window.
  16. Under the Options menu at the top of the screen select Rebuild.
  17. Save your level!
  
 

 

Adding a Sound to Your Door
 
  1. Right Click on the edge of your mover and select Mover Properties.
  2. In the Mover Properties window select MoverSounds.
  3. In the Browse window select SoundFX.
  4. In the Browse window beneath SoundFX select General.
  5. Select Load at the bottom of the Browse window.
  6. Select DoorsAnc.uax and click Open.
  7. In the Browse window beneath DoorsAnc select Stone.
  8. In the Browse window select sdoorS4.
  9. In the Mover Properties window select ClosingSound.
  10. In the Mover Properties window next to ClosingSound select Use.
  11. In the Browse window select sdoorS1.
  12. In the Mover Properties window select OpeningSound.
  13. In the Mover Properties window next to OpeningSound select Use.
  14. Close the Mover Properties window.
  15. Under the Options menu at the top of the screen select Rebuild.
  16. Save your level!
  
 

 

Adding Music to your Level
 
  1. In the Browse window select Music.
  2. Select Load at the bottom of the Browse window.
  3. Select EverSmoke.umx and click Open.
  4. Under the Options menu at the top of screen select Level Properties.
  5. In the Level Properties window select Audio.
  6. In the Level Properties window select Song.
  7. In the Level Properties window next to Song select Use.
  8. Close the Level Properties window.
  9. Under the Options menu at the top of the screen select Rebuild.
  10. Save your level!
  11. Select Play Level under the File menu at the top of the screen.
