PART~2
Adding Sound


Ok, now it's time to make a little sound to make that door come to life ey?.....
Let us start by going back into unrealED and "right click" that mover brush and select "Mover Properties" and go to "Mover" then to Mover Sounds", select the "ClosedSound" and there is a button with "...", click it!



now on the right in the "browse"section the "SoundFX" loaded.
Now, click the "Load" button at the bottom and this will bring up a dialog box like this.....



Choose the "DoorsAnc.uax" file and open it.
This is now in your Browse section. At the third pull down menu select "GenericThumps" and go down to the list below and select "adend61"



Now back in your Mover Properties box at the "ClosedSound", hit the "USE" button, the sound should be labled in that line!
Now below that go to MoveAmbientSound". Now go back to the Browse menu and the third pull down menu, select "Stone" and go down to the list and select "stnloop4".
Now go back to the Mover Properties and at the MoveAmbientSound, click the "use" button....


one more, you still with me here?...ok


the last but not least is the "OpeningSound" so select it and go back over to the Browse menu and the third pull menu, select "GenericThumps" again and in the list you'll see "adend56", select it and go back over to the Mover Properties box and click the use button at the OpeningSound line,......
