Purpose of the S3TC server packages
 

The sole purpose of these downloads and explanations is to make it possible for S3TC players to go play online without having to deal with the so called "skating player glitch" We asume that any server admin wants to make sure clients can play on their server(s) without any problems or hassle. If you are a server admin than you can help your clients by switching out all Stock Unreal Tournament textures for the textures offered here.

In effect it doesnt change anything at all on the server, the gameplay will be the same, the mods will be the same, the maps will be the same, the downloads will be the same(which is none since the packages are stock packages), etc, etc.

The only thing it does, is making sure S3TC players will have just as much fun playing on your server(s) as any other player. So if you want to help out the Unreal Tournament community than its highly appreciate if you go use the serverpackages for your server(s).








What do you need to do ?
 

This is a simple explanation really, for Servers you have to use the normal textures! The S3TC textures are purelly ment for ClientSided playing and should never be used for Servers nor Editing !

Also, all textures that have been released as so called Net textures (reduced 'server only' textures) should never be used !








Skating player Glitch
 

What is the skating player glitch ?

The skating player glitch occurs when players use the S3TC textures for online playing. If the server is loaded with the improper textures than the glitch will apear. From the players point of view all other players and bots apear to be frozen and seem to skate around on oné leg. If such a player is killed, a ghost image from the dead player remains in view (garbage is not being collected and destroyed).




Anyone can understand that this is an unwanted situation, and, actually pretty easy to solve...........

- Server admins should never, ever load any S3TC textures on their server(s) !
- Server admins should never use the so called Net textures (reduced 'server only' textures) !
- Server admins should never use the textures from the 2nd CD !




Solution:

Serveradmins should only use the normal textures that came with the 1st CD.








Downloads
 

If needed you can download the texture packages that are safe to use for servers. The downloads contain all texture packages from the 1st CD as well as 5 texture packages from the 2nd CD that aren't present on the 1st CD. The downloads also contain the texture packages from the Bonus Packs that cannot be found on the CD's. Than there is an additional UnrealGold package in the download. Loading these textures on the server should cover all textures released by Epic for Unreal, UnrealGold, Unreal Tournament and Unreal Tournament G.O.T.Y.

For the downloads you can choose the platform you use to run Unreal Tournament or any filetype you wish. All downloads contain the exact same texture packages, simply pick the filetype of your choice.

Installation

Anthology Unreal Tournament:

Add the textures to your C:\Unreal Anthology\UnrealTournament\Texture folder by overwriting the excisting packages inthere.

Q and A
 

---------------------------------
Q: If i use these packages, will clients have to download those from my server ?
A: No, these packages are stock textures from UT, and all clients already have them in their Texture folder.


.---------------------------------
Q: Does that also apply to the extra packages like the UnrealGold and the mod packages ?
A: No, if a map is run that needs the mod or UnrealGold textures, the clients will download those from your server, providing they dont have them. But effectivelly this is what always happens if you run custom maps and mods on your server. So for those its business as usual.


.---------------------------------
Q: Will the behaviour of the server change ?
A: No, you will not notice any difference.


.---------------------------------
Q: Will the behaviour of the mods change ?
A: No, they will work the same as before, nothing different will happen.


.---------------------------------
Q: Is there any change at all ?
A: No-one will know there you are using "other" textures, only S3TC clients will notice they now CAN play on your server.
